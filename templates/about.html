About Walker Files Blog

Welcome to My Cybersecurity Blog

Walker Files Blog is a secure, feature-rich blogging platform built as part of my web development journey at Dunwoody College of Technology. This project showcases modern web development practices with a focus on security and user experience.

What We Offer

This blog provides a platform for sharing insights about cybersecurity, web development, and technology. With a sleek dark cyberpunk aesthetic and robust security features, it demonstrates professional-grade web application development.

Technology Stack

Built with Flask (Python web framework), SQLAlchemy for database management, and Flask-Login for secure authentication. Input sanitization is handled by the Bleach library to prevent XSS attacks while preserving safe HTML formatting.

Key Features

  • Secure Authentication: Password hashing and session management
  • Role-Based Access: Admin-controlled content creation
  • Input Sanitization: Protection against XSS attacks
  • User Management: Admin dashboard for user oversight
  • Interactive Comments: Community engagement on posts
  • Responsive Design: Works beautifully on all devices

About the Developer

I'm Will Walker, a cybersecurity student at Dunwoody College of Technology. Originally starting in computer engineering, I found my passion in cybersecurity and made the switch to focus on building secure, user-friendly applications that prioritize both functionality and security best practices.

Project Goals

This blog was created to demonstrate:

  • Full-stack web development capabilities
  • Security-first development practices
  • Database design and relationships
  • User authentication and authorization
  • Modern UI/UX design principles
  • Comprehensive testing methodologies
